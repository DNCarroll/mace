mace.js Objects

ViewContainer

associates a given url via its name but can be customized, determines if it is the default view for a given area, and supplies Views (HTML fragments) that are associated to html container elements. More on ViewContainers can be found here.
Required.

View

associates to HTML fragment by name but can be customized, determines cache strategy for the HTML fragment, and what target content element will recieve the HTML fragment. More on Views can be found here.
Optional.

Binder

associates HTML Elements to a WebApi Controller. Inheritance allows for customization. Utilizes GET, POST, PUT, DELETE methods. Binds data returned from GET method to the associated HTML Element. More on Binders can be found here.
Required (normally declared in HTML Fragment).

DataObject

All incoming data from GET is stored in this base object. Inheritance allows for customization. More on DataObjects can be found here.
Optional.
