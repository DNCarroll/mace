Steps for constructing with ULFBERHT

not in any particular order
  1. build any html views that will be used by a ViewContainer
  2. build View or classes inheriting from IView
  3. build the View container using the views above
    1. make sure that in the constructor you add them to the Array<View>, 'Views'
    2. add the ViewContainer to the ViewContainers eg ViewContainers.push(new DefaultContentManager());
    3. insure that one ViewContainer is the Default (IsDefault)
  4. if testing and adding each js file individually make sure to add the View.js's first then the Containers
